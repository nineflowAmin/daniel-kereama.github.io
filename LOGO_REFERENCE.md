## Logo Specifications

- **Format**: SVG preferred (scalable), PNG/JPG acceptable
- **Style**: Monochrome/grayscale versions work best for the ticker
- **Size**: Max height 50px, max width 150px (maintain aspect ratio)
- **Background**: Transparent preferred
- **Color**: Logos will be displayed in grayscale by default, color on hover

## Where to Find Official Logos

1. **Company Websites**: Most companies have brand asset pages
   - Look for "Media", "Press", "Brand Assets", or "Logo Downloads"

2. **Brand Guidelines Pages**:
   - Equifax: Check Equifax brand guidelines
   - Domino's: Domino's brand resources
   - Rio Tinto: Rio Tinto brand center
   - Spark NZ: Spark brand guidelines

3. **Logo Databases**:
   - Simple Icons
   - World Vector Logo
   - Brandfetch
   - Clearbit Logo API

4. **Contact Companies**: For smaller companies (Automated Electrics, Pipefish), you may need to contact them directly for logo assets

## Fallback Behavior

The ticker includes fallback text that displays if logo images are not found. The code automatically shows company names if logos fail to load.

## Current Status

The logo ticker is configured to use images but will gracefully fall back to text if logos are not available. Once you add the logo files to `assets/images/logos/`, they will automatically appear in the ticker.
